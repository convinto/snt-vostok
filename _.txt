📚 Руководство по работе с админ-панелью СНТ «ВОСТОК»
Вход в админ-панель
Откройте браузер и перейдите по адресу: http://ваш-сайт.ru/admin

Введите логин (email) и пароль администратора, которые вам выдал предыдущий председатель

Нажмите «Войти»

Важно: Доступ к админ-панели есть только у председателя и доверенных лиц с правами администратора.

Обзор главного экрана
После входа вы увидите список разделов:

Раздел	Назначение
User	Управление пользователями (садоводами)
Plot	Реестр участков
Tariff	Тарифы на электроэнергию
Meter Reading	Показания электросчётчиков
Payment	Учёт платежей
Post	Новости и объявления для сайта
1. Раздел «User» — Пользователи (садоводы)
Здесь хранятся все зарегистрированные садоводы.

Что вы видите в таблице:
Full Name — ФИО садовода

Username — логин для входа

Email — электронная почта

Phone — телефон

Desired Plot Number — участок, который указал садовод при регистрации

Is Admin — галочка, если у человека есть права администратора

Plot — привязанный участок (если есть)

Как найти садовода:
В верхней части таблицы есть строка поиска. Введите ФИО, телефон или номер участка — система покажет совпадения.

Как добавить нового садовода:
Нажмите кнопку Create вверху таблицы

Заполните обязательные поля:

Username — придумайте логин (например, фамилия латиницей)

Email — электронная почта садовода

Full Name — полное ФИО

Phone — номер телефона

В поле New Password задайте временный пароль

Нажмите Save

Как отредактировать данные:
Нажмите на строку с садоводом (или на иконку карандаша)

Измените нужные поля

Нажмите Save

Как сбросить пароль садоводу:
Способ 1 — вручную:

Нажмите на строку садовода

В поле New Password введите новый пароль

Нажмите Save

Сообщите новый пароль садоводу лично или по телефону

Способ 2 — массовый сброс:

Отметьте галочками нужных пользователей

В выпадающем списке «С действиями» выберите «Сбросить пароль»

Нажмите Выполнить

Система покажет новые пароли — запишите или сфотографируйте их, чтобы передать садоводам

Как привязать участок к садоводу:
Нажмите на строку садовода

В поле Plot выберите нужный участок из списка

Нажмите Save

Подсказка: Перед привязкой убедитесь, что участок уже создан в разделе «Plot».

2. Раздел «Plot» — Участки
Это реестр всех участков СНТ.

Что вы видите в таблице:
Plot Number — номер участка

Cadastral Number — кадастровый номер (если есть)

Area — площадь в сотках

Owner Name — ФИО владельца по документам

Owner — привязанный пользователь

Electricity Tariff — не используется (тарифы теперь в разделе Tariff)

Membership Fee — размер членского взноса

Как добавить новый участок:
Нажмите Create

Заполните поля:

Plot Number — номер участка (обязательно)

Cadastral Number — кадастровый номер (если есть)

Area — площадь в сотках

Owner Name — ФИО владельца

Membership Fee — размер членского взноса за год (по умолчанию 1500 руб.)

Нажмите Save

3. Раздел «Tariff» — Тарифы на электроэнергию
Здесь хранится история тарифов. Тарифы могут меняться 1-2 раза в год, и система запоминает все изменения.

Как работают тарифы:
Base (Гос. тариф) — официальный тариф, установленный государством

Losses (Потери в сетях) — дополнительный процент на покрытие потерь в сетях СНТ

Общая стоимость = Гос. тариф + Потери в сетях.

Что вы видите в таблице:
Tariff Type — тип тарифа (Base или Losses)

Rate — ставка в рублях за 1 кВт⋅ч

Effective From — дата, с которой действует тариф

Description — основание (номер и дата приказа)

Как добавить новый тариф:
Нажмите Create

Выберите Tariff Type:

base — если меняется государственный тариф

losses — если меняется процент на потери

В поле Rate укажите новую ставку в рублях за 1 кВт⋅ч

В поле Effective From укажите дату, с которой тариф начинает действовать

В поле Description напишите основание (например: «Приказ ФАС №123 от 01.01.2026»)

Нажмите Save

Важно: Указывайте реальную дату начала действия. При расчёте показаний система автоматически подставит тариф, действовавший на дату снятия показаний.

Когда менять тарифы:
При получении официального уведомления об изменении государственного тарифа

При решении общего собрания об изменении процента на потери

Создавайте новую запись — старая сохранится в истории

4. Раздел «Meter Reading» — Показания счётчиков
Здесь учитывается расход электроэнергии по каждому участку.

Что вы видите в таблице:
User — садовод

Plot — номер участка

Period — период (ГГГГ-ММ)

Reading Value — текущее показание счётчика

Consumption — расход за период (рассчитывается автоматически)

Base Amount — сумма по гос. тарифу

Losses Amount — сумма на покрытие потерь

Total Amount — итоговая сумма к оплате

Is Verified — подтверждено председателем

Как добавить показания:
Нажмите Create

Выберите User — садовода

Выберите Plot — его участок

В поле Reading Value введите текущее показание счётчика (только цифры)

В поле Reading Date выберите дату снятия показаний

В поле Period введите период в формате ГГГГ-ММ (например, 2026-05)

Нажмите Save

Система автоматически:

Найдёт предыдущее показание этого участка

Рассчитает расход (разницу между текущим и предыдущим показанием)

Возьмёт актуальные тарифы на дату показаний

Рассчитает сумму по гос. тарифу, сумму на потери и итог

Как подтвердить показания:
Нажмите на строку с показанием

Поставьте галочку Is Verified

Нажмите Save

Подтверждённые показания будут видны садоводу в личном кабинете.

Как выгрузить данные в Excel:
Отметьте галочками нужные строки

В выпадающем списке выберите «Выгрузить в CSV»

Нажмите Выполнить

Файл сохранится на компьютер. Его можно открыть в Excel

5. Раздел «Payment» — Платежи
Здесь учитываются все платежи садоводов: членские взносы и оплата электроэнергии.

Что вы видите в таблице:
User — садовод

Plot — номер участка

Payment Type — тип платежа

Amount — сумма

Payment Date — дата платежа

Period — за какой период

Verified By Admin — подтверждён ли платёж

Типы платежей:
membership — членские взносы

electricity — электроэнергия

target — целевой взнос

Как добавить платёж вручную:
Нажмите Create

Выберите User — садовода

Выберите Plot — его участок

В поле Payment Type выберите тип платежа из списка

В поле Amount укажите сумму в рублях

В поле Payment Date укажите дату платежа

В поле Period укажите период (например, 2026 для годового взноса или 2026-05 для помесячного)

В поле Description напишите примечание (например, «Оплата по QR-коду от 15.05.2026»)

Поставьте галочку Verified By Admin, если платёж подтверждён

Нажмите Save

Как подтвердить платёж:
Получив уведомление из банка или от садовода, найдите платёж в таблице

Нажмите на строку

Поставьте галочку Verified By Admin

Нажмите Save

Подтверждённые платежи отображаются в личном кабинете садовода с зелёной пометкой «Подтверждён».

Как выгрузить данные в Excel:
Отметьте галочками нужные строки

В выпадающем списке выберите «Выгрузить в CSV»

Нажмите Выполнить

Файл откроется в Excel

6. Раздел «Post» — Новости и объявления
Здесь создаются новости, которые видны всем садоводам на главной странице.

Как создать новость:
Нажмите Create

В поле Title введите заголовок

В поле Body введите текст новости

В поле Author выберите себя (администратора)

Нажмите Save

Новость сразу появится на главной странице сайта.

Как отредактировать или удалить новость:
Нажмите на строку с новостью

Измените текст — нажмите Save

Чтобы удалить — нажмите на иконку корзины и подтвердите удаление

Типичные сценарии работы
🔹 Приняли нового садовода в СНТ:
Зайдите в User → Create

Создайте пользователя, заполнив все данные и задав временный пароль

Если участок уже есть в системе — привяжите его в поле Plot

Если участка нет — создайте его в разделе Plot, затем привяжите

Передайте садоводу логин и пароль

🔹 Садовод прислал фото счётчика:
Зайдите в Meter Reading → Create

Выберите садовода и участок

Введите показание с датой и периодом

Система сама рассчитает расход и сумму

Поставьте галочку Is Verified

Садовод увидит данные в личном кабинете

🔹 Садовод оплатил взнос или свет:
Получите подтверждение оплаты (выписка из банка, скриншот)

Зайдите в Payment → Create

Выберите садовода, участок, тип платежа, сумму и дату

Поставьте галочку Verified By Admin

Платёж отразится в личном кабинете садовода

🔹 Изменился тариф на электроэнергию:
Зайдите в Tariff → Create

Выберите тип тарифа (base или losses)

Укажите новую ставку

Укажите дату начала действия

Напишите основание (номер приказа)

🔹 Нужна выгрузка для ревизии:
Зайдите в нужный раздел (Payment или Meter Reading)

При необходимости отфильтруйте записи по периоду или участку

Отметьте все нужные строки галочками

Выберите «Выгрузить в CSV»

Откройте файл в Excel

🔹 Садовод забыл пароль:
Зайдите в User

Найдите садовода (поиск по ФИО или телефону)

Нажмите на строку

В поле New Password введите новый пароль

Нажмите Save

Сообщите пароль садоводу

Важные замечания
Не удаляйте старые записи. История платежей и показаний важна для отчётности. Лучше используйте фильтры для просмотра нужных периодов.

Подтверждайте данные. Садоводы видят в личном кабинете только подтверждённые платежи и показания. Неподтверждённые записи помечаются как «В обработке».

Следите за актуальностью тарифов. При изменении тарифов сразу создавайте новую запись в разделе Tariff. Система автоматически применит правильный тариф к показаниям на соответствующую дату.

Экспорт для собраний. Перед общим собранием выгрузите данные по платежам — это поможет ответить на вопросы садоводов о задолженностях.
